# Como executar

## SIREN
1. Baixe/clone este repositório.
2. Abra o notebook **`SIREN_Nova_Implementacao.ipynb`**.
3. Execute as células **passo a passo**.

## Outros Spatial Encoders
1. Abra a pasta **`Location Encoders/`**.
2. Escolha o notebook `.ipynb` do encoder desejado.
3. Execute as células **passo a passo**.
